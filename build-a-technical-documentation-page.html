Introduction

Hyper Text Markup Language, or simply HTML, is a standard markup language for Web pages. It's used to describe the structure of Webpages. It's a very simple language to learn and a good way to start for those who are giving the first steps in the technology area.

The syntax of HTML consists in elements that wrap pieces of content. Those are called tags:

  • <b>bold</b>
  • <i>italic</i>
  • <i>underline</i>

We'll se other examples in this documentation.

The Basics

All HTML document should start with an instruction that states it's a HTML file:

<!DOCTYPE html>

Note that there is a ! in the beginning of the information inside the tags. This sign indicates a comment. This means that only the browser and developers can see this. It'll not appear in the Webpage.

Other essencial tags a HTML code should have:

  • <link rel="stylesheet" href="css_sheet"> - tell the browser where to look for instructions for the page's style
  • <meta> - adds information about metadata within the HTML document
  • <body></body> - encloses all the contents of the HTM document
    •

There are other important tags in a HTML document, but those were just a few examples of what we can expect to find on the Websites we usually visit.

Headings

Headigns on HTML goes from 1 (the biggest) to 6 (the smallest). In the beginning, it seems a little weird - how come 1 is bigger than 6? But it helps if we think about it as a ranking, where 1 representes the most important heading, and 6, the last one.

  • Heading 1

  • Heading 2

Headings are great for organizing the content of the Website and showing the search engines what is most important in the page. If you want to learn more about that, you may be interested in looking for some information on SEO.

One Step Ahead

HTML alone is not able to give a Webpage some style. It only works as the structure behind that beautiful website we're intending to build. So, how do we give it a some styling?

To do that, we'll have to go a little further in our studies and use another tool: CSS. CSS will let us make that website more appealing, interesting and assessible. We can add CSS instructions to a HTML document in 3 ways:

  • style atribute - the instruction is added directly to the tag inside it, with the style atribute
  • <style> element - the instruction is added with a pair of tags of style
  • <link> element - all the CSS instructions for a HTML document are created in a separate document, which is acessed but the HTML document through a <link> element
